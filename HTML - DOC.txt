Para CSS:
podemos usar id e class na mesma tag(<.. id="exemplo01" class"exemplo02">)
id    | (Prioritário)identificação para o CSS(o nome não se repete)
class | identificação para o CSS(o nome pode se repetir)
    pode usar mais de um na mesma tag separando por espaço(
        <.. class="exemplo01 exemplo02">
    )

Listas:

ul li | lista bolinha(TIpos: 1, A, a, I, i) (definir a lista)
ol li | lista numerada(Tipos: •disc, ∘circle, ∎square) (definir a lista)
    li | lista com demarcadores (listar o item)
dl | lista com definição
    dt | Proposta  
        dd | definição


Links(a):

a ou a com target="_self"| link (Sobrepõe a página)
a com target="_blank" | abre link em nova aba
    "rel" em "a" é para o buscador se guiar (
        next | proxima pagina
        prev | pagina anterior
        nofollow | não quero que o link seja relacionado ao meu site
        )
a com download="endereço do aquivo" type="" | Link para Download (Consultar tipos em https://www.iana.org/assignments/media-types/media-types.xhtml)


Imagens:

picture | obj imagem(Permite modificação por tamanho de tela)
<source media="(max-width: tamanho em px)" srcset="endereço_imagem" type="tipo_imagem">
colocar em ordem descrescente e ultimo chamada normal(img)
em "max-width" é interessante colocar +- 50px a mais!


Áudio: (lembrar de usar o "controls" para aparecer os controles)

Formatos de áudio suportados: MP3 WAV OGG
audio | obj audio(Ordme de tentativa de reprodução, permite mensagem se não conseguir reproduzir)
<source src="endereço_audio" type="tipo_audio">


Vídeos: (lembrar de usar o "controls" para aparecer os controles)

Dica de programa de conversão: HandBrake
o ideal é ultilizar vídeos externos para não consumir banda do site
video | obj video(Permite modificação por tamanho de tela)
<source src="endereço_video" type="tipo_video">
